# zed-just

[Just](https://github.com/casey/just) language support for the [Zed](https://zed.dev) editor.

## Features

- Syntax highlighting via tree-sitter
- Language server support with [just-lsp](https://github.com/terror/just-lsp)
  - Hover documentation
  - Diagnostics
  - Formatting
- Run recipes directly from the editor

The LSP server is automatically downloaded if not already installed.
